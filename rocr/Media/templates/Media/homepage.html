ROC curve analysis

Suppose we have several (original) receiver operating characteristic (ROC) curves. We want to combine all these ROC curves together using some "rule" to produce a new (derived) ROC curve. There can be several different combining rules, thus, there can be several derived ROC curves.

The Goal of this software is to take in two or more ROC functions, input a combining rule, then output the derived ROC curve.
